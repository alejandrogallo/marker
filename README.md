# MarkerFork
Something similar to smenu but in python.

## Requirements
- python (2.7+ or 3.0+)

## Usage

```python
#! /usr/bin/env python
# -*- coding: utf-8 -*-
import marker.core
import marker.renderer

a = marker.core.pick('', ['Item 1', 'Item 2'])
print(a)
```
